# Introduction

- Floating-points Numbers  are a specific data type in C that allows representation of real numbers including decimal fractions. <br>
- The type cast Operator is a powerful tool for explicity converting one data type into another.<br>
- The type char represents character data in C. It is a fundamental data type for handling single character.<br>
- Keywords in C are reserved words that have predefined meanings. These words are crucial for constructing statements and expressions in a C program.<br>
