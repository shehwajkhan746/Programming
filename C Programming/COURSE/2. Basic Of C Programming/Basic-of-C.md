## 2.1 Floating-point Numbers
Floating point (or real) numbers are stored in 32 bit (on all 16 bit and 32 bit machines), with 6 digit of precision. Floating point numbers are defined in C by the keyword float. <br>
A double data type number uses 64 bits giving a precision of 14 digits. These are known as double precision numbers.
